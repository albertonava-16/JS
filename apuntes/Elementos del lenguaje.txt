Tipos de datos en JavaScript

1.- Tipos primitivos

 * Undefined: tipo especial, contiene undefined

2.- Tipos Objeto
 * Tipos predifinidos de JS
   - Date (fechas)
   - RegExp (expresiones regulares)
   - Error (datos de error)

 * Tipos definidos por el programador/Usuario
   - Funciones simples
   - Clases

 * Arrays 
   - Serie de elementos o formaciòn tipo vector o matriz. Lo      consideraremos un objeto especial.

  * Objetos especiales
   - Objeto global
